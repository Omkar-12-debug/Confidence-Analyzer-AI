# Bhavesh's Visual Processing Module

## Multimodal AI-Based Confidence and Stress Level Analysis

### Overview
This module handles all visual processing tasks for the project, including face detection, landmark extraction, emotion recognition, and behavior metrics calculation.

### Features
- Real-time face detection using MediaPipe
- Facial landmark extraction (468 points)
- Emotion recognition using FER2013 pretrained model
- Behavior metrics calculation:
  - Blink rate detection
  - Eye contact percentage
  - Head movement frequency
  - Emotion stability analysis
- Video recording and frame extraction
- Feature export for fusion module
